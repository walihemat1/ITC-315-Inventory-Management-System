### Product Vision
"Our Inventory Management System helps small shop owners manage products, track stock levels, generate basic reports, and receive low-stock alerts easily and efficiently."

Initial Product Backlog with 8 User Stories 
# User Story 1: Add New Product
As a shop owner, I want to add new products so that I can keep track of items in my store.
# User Story 2: Edit Product Information
As a shop owner, I want to update product name, category, and price so the information stays accurate.
# User Story 3: Update Product Quantity
As a shop owner, I want to adjust stock levels when new items arrive or are sold so that inventory remains correct.
# User Story 4: Delete a Product
As a shop owner, I want to delete products that I no longer sell so that my inventory list stays clean.
# User Story 5: View Product List
As a shop owner, I want to see all products in one list so that I can monitor inventory easily.
# User Story 6: Low-Stock Alert
As a shop owner, I want to get alerts when a product is running low so that I can reorder on time.
# User Story 7: Search Products
As a shop owner, I want to search for products by name or category so that I can find them quickly.
