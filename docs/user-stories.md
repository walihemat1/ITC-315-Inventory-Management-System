Inventory Management System
Product Vision
"Our Inventory Management System helps small shop owners manage products, track stock levels, generate basic reports, and receive low-stock alerts easily and efficiently."

User Stories
1. Product Management (8 Stories)
US1: As an admin, I want to add a new product so that it appears in the system.
US2: As an admin, I want to edit product details so that information stays correct.
US3: As an admin, I want to delete a product so that old items are removed.
US4: As a user, I want to view a list of all products so that I can see inventory items.
US5: As a user, I want to search products by name so that I can find items quickly.
US6: As a user, I want to search products by SKU so that I can locate specific items.
US7: As an admin, I want to upload a product image so that items are visually identifiable.
US8: As an admin, I want to categorize products so that they are more organized.
2. Stock Management (10 Stories)
US9: As a staff member, I want to record stock-in so that inventory quantity increases.
US10: As a staff member, I want to record stock-out so that inventory quantity decreases.
US11: As a user, I want to see current stock levels so that I know what is available.
US12: As a user, I want the system to automatically update stock after stock-in/out so that accuracy is maintained.
US13: As a staff member, I want to enter notes (e.g., damaged, expired) during stock-out so that the reason is clear.
US14: As a manager, I want to set a minimum quantity level for products so that I know when stock is low.
US15: As a user, I want to see a "Low Stock" badge on items below minimum threshold so that I can act quickly.
US16: As a staff member, I want to correct stock quantity manually so that errors can be fixed.
US17: As a user, I want to view stock-in history so that I can review past additions.
US18: As a user, I want to view stock-out history so that I can track outgoing items.
3. User Authentication & Roles (6 Stories)
US19: As a user, I want to register an account (optional approval) so that I can access the system.
US20: As an admin, I want to log in securely so that only authorized users access the system.
US21: As an admin, I want to log out so that I can end my session.
US22: As an admin, I want to create staff accounts so that others can manage stock.
US23: As an admin, I want to assign roles (Admin/Staff) so that permissions are controlled.
US24: As an admin, I want to deactivate a user account so that inactive users cannot access the system.
4. Dashboard (5 Stories)
US25: As a user, I want to see the total number of products on the dashboard so that I can understand inventory size.
US26: As a user, I want to see total stock-in for today so that I know how much inventory arrived.
US27: As a user, I want to see total stock-out for today so that I know how many items were used/sold.
US28: As a user, I want to see a list of low-stock items on the dashboard so that I can restock urgently.
US29: As a user, I want a simple chart showing stock-in vs stock-out so that I can visualize activity.
5. Reporting (4 Stories)
US30: As a manager, I want to generate a low-stock report so that I can reorder items.
US31: As a manager, I want to generate a stock movement report so that I can track system activity.
US32: As a manager, I want to export reports to PDF so that they can be shared.
US33: As a user, I want to filter reports by date range so that I can view specific periods.
6. General UI/UX Enhancements
US34: As a user, I want the system to be fully responsive so that I can use it on all devices.
US35: As a user, I want clear success/error notifications so that I understand system actions.
